# nixos-package-search

An extension for .nix files, to avoid annoying package lookup

## Features

Automatically searches for nixos packages when you start typing

## Requirements

nix-search

## Extension Settings

No settings are currently available

## Known Issues

No known issues

## Release Notes

If you find any bugs feel free to make a pull request on github
---

## Installation
You will require vsce for the installation
Simply run:
```
vsce package
code --install-extension [name of generated .vsix file]
```

**Enjoy!**
